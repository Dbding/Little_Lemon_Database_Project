# Tableau Analysis Report

## Dashboard title
Little Lemon Sales and Bookings Dashboard

## Data source
Use the provided LittleLemon_data.xlsx file or connect directly to the MySQL database.

## Suggested worksheets
1. Sales by menu/cuisine
2. Booking count by date
3. Orders by quantity
4. Delivery status summary

## Dashboard design
The dashboard should include:
- Bar chart for sales by menu or cuisine
- Line chart or table for booking activity
- Bar chart for order quantity
- Pie or bar chart for delivery status

## Purpose
The dashboard helps Little Lemon review sales, bookings, order volume, and delivery performance.
